# Ecommerce Backend

A Django-based eCommerce backend project with authentication, order management, payments, coupons, wishlist, and seller features.

## Features

- User Authentication
- Product Management
- Order System
- Wishlist Feature
- Payment Integration
- Coupon System
- Seller Management
- REST API Support

## Technologies Used

- Python
- Django
- SQLite
- REST APIs
- Git & GitHub

## Project Structure

- users/ → User authentication & profiles
- orders/ → Order management
- payments/ → Payment handling
- wishlist/ → Wishlist functionality
- sellers/ → Seller dashboard
- coupons/ → Coupon management

## Installation

```bash
git clone https://github.com/yourusername/ecommerce-backend.git
cd ecommerce-backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
